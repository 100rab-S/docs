# Install Dependencies on your computer

## Install Package Manager

On Linux

```
# update
sudo apt-get update
```

### On MacOS

install package manager if it is not installed yet
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

update it
```
brew update
```
